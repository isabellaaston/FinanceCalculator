The upfront course fee for the Full Stack Track is £8,000* (*£10k from January 2021), however, depending on how much you borrow, the total fee can change.

EdAid charge a 5% admin fee on any money borrowed.

e.g. If you borrow £6,000, you pay a £300 admin fee upfront. This does not go towards the cost of the course, and is a fee charged by EdAid, not Mayden Academy.
  • If you borrow up to 80%, the course fee is £8,000 (+ 5% EdAid admin fee)
  • If you borrow 81% – 90%, the course fee is £8,500 (+ 5% EdAid admin fee)
  • If you borrow over 90%, the course fee is £9,000 (+ 5% EdAid admin fee)
